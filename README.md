# Contact Manager Application

## Prerequisites

- Go(version **1.12.9**)
- Postgresql

## Get started

- Clone this repository:
```
git clone https://github.com/topd333/golang-restful-api.git
```

- Configure environment
```
cd golang-restful-api
cp .env.example .env
```
Please fill database information to `.env` file

- Run the server
```
go run main.go
```
